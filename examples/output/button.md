# button

Base action button

## Usage

```html
<cj-button variant="">
  <span slot="iconLeft">...</span>
  <span slot="label">...</span>
  <span slot="iconRight">...</span>
</cj-button>
```

## Variants

- `sm`
- `lg`

## States

- `:hover`
- `:active`

## Slots

- `iconLeft`
- `label`
- `iconRight`
